🎛️ Virtual Volume Controller Using Hand Gestures

This Python project uses MediaPipe and Pycaw to control system volume via hand gestures detected through a webcam.

🔧 Technologies Used

  • Python
  • MediaPipe
  • OpenCV
  • Pycaw

📦 How It Works

It tracks hand landmarks using MediaPipe, calculates the distance between fingers, and maps that to system volume using Pycaw.

🚀 Try It Locally

pip install mediapipe opencv-python pycaw
python volume_controller.py
